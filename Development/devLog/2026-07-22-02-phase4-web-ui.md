## 技术决策

### 依赖策略

无新增 npm 依赖。react-router-dom v7 已装，CSS Modules 内置于 Vite。Markdown 渲染手写正则（ponytail：覆盖诊断报告格式足够），不引入 `marked`/`DOMPurify`。

### API 客户端

- `request<T>()` — 单条数据，解包 `ApiResponse<T>` → `T`，处理 204/401/ApiError
- `getList<T>()` — 列表数据，直接返回 `ApiListResponse<T>`（后端专用包装类型）
- `connectSse()` — SSE 流，`ReadableStream` 逐行解析，回调模式

### 状态管理

无 Redux/Zustand。每个页面独立 `useState` + `useEffect`，loading/error/data 三态一致。SSE 数据通过回调更新 React state。
