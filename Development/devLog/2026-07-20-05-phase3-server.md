## 技术决策

1. **SecurityFilter 链顺序**：RateLimit→Session→Bearer。限流最外层减少无效认证开销；Session 优先 Bearer（Web UI 场景）
2. **BearerToken 认证主体简化**：ApiToken 实体缺 getAdmin() getter，认证主体用字符串 "admin" 代替 Admin 对象。ponytail: 单管理员场景下足够
3. **SseEmitterManager 实现 DiagnosisSseEventPublisher**：不需要额外适配层，直接注入给编排器
4. **API 响应统一包装**：ApiResponse<T>(data, error) + ApiListResponse<T>(data, total)
5. **ConversationController.sendMessage 同步阻塞**：Virtual Threads 环境下同步=异步，无需 @Async
6. **Controller 桩实现**：probe/test 端点返回占位成功响应，真实实现在 Phase 4/5 补充
