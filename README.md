# Epiphaneia

面向后端开发者的 AI Agent 智能诊断工作台。从想法到上线的全流程管理仓库。基于 Java Spring 生态。

## 流程节点

| 阶段 | 说明 |
|------|------|
| IdeaValidation | 想法验证 — 市场调研、竞品分析、用户需求确认 |
| ProductPlanning | 规划设计 — PRD、领域建模、API设计、数据库设计 |
| ArchitectureDesign | 架构设计 — 技术选型、系统架构、安全设计 |
| Development | 开发实现 — 编码、单元测试、代码审查 |
| Testing | 测试阶段 — 集成测试、性能测试、验收测试 |
| ReleaseDeploy | 发布上线 — CI/CD、环境配置、灰度发布 |
| PostLaunch | 上线后 — 监控告警、用户反馈、迭代规划 |

## 技术栈

- 后端：Java Spring 生态
- 具体技术选型在 [ArchitectureDesign](ArchitectureDesign/) 阶段确定
